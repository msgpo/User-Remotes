# MPC-BE
Media Player Classic Black Edition Remote.

## Features
*  Playback (play, pause, stop)
*  Track (next, previous)
*  Volume (up, down, mute)
*  Navigate (up, down, left, right, select, back, home)
*  Seek (forward, rewind)
*  Jump 5s (back, forward)
*  Open (chapters, titles)
*  Toogle Fullscreen
